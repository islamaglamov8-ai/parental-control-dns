# 🛡 Родительский контроль (DNS фильтр)

Программа для ограничения доступа к сайтам через локальный DNS-сервер.

---

# ⚙️ Требования

## 🐍 Python

* Версия: **Python 3.10 – 3.12**
* Рекомендуется: **Python 3.11**

Проверить версию:

```bash
python --version
```

---

# 📦 Зависимости

Установи необходимые библиотеки:

```bash
pip install dnslib psutil pyinstaller
```

---

# 🚀 Запуск программы (из Python)

1. Открой терминал (cmd / PowerShell)
2. Перейди в папку проекта:

```bash
cd путь_к_папке
```

3. Запусти:

```bash
python parental_control.py
```

⚠️ ВАЖНО: запускать **от имени администратора**

---

# 🧱 Сборка в .exe

## Установка сборщика:

```bash
pip install pyinstaller
```

## Сборка:

```bash
pyinstaller --onefile --noconsole --uac-admin parental_control.py
```

## Результат:

```text
dist/
└── parental_control.exe
```

---

# 💻 Запуск на другом ПК

1. Скопируй файл:

```text
parental_control.exe
```

2. Запусти:

* двойной клик
* подтвердить запрос администратора (UAC)

---

# 🔧 Использование

## В программе:

* ➕ Добавить сайт — добавить в whitelist
* ➖ Удалить сайт — удалить из whitelist
* ▶ Запустить DNS — включить фильтр
* ⏹ Остановить DNS — выключить фильтр
* 🌐 Вернуть интернет — восстановить настройки

---

# 🔍 Проверка работы

Открой терминал:

```bash
nslookup ya.ru
nslookup google.com
```

### Ожидаемый результат:

| Сайт                     | Результат     |
| ------------------------ | ------------- |
| ya.ru (в списке)         | нормальный IP |
| google.com (не в списке) | 0.0.0.0       |

---

# ⚠️ Важные замечания

## 1. Требуются права администратора

Без них DNS работать не будет.

---

## 2. Возможные проблемы

### ❌ Интернет не работает

```bash
netsh interface ip set dns name="Wi-Fi" source=dhcp
ipconfig /flushdns
```

---

### ❌ Порт 53 занят

```bash
net stop dnscache
```

---

### ❌ Браузер обходит фильтр

Отключи:

* DNS over HTTPS (DoH)
* VPN / прокси

---

# 🧠 Как это работает

1. Запускается локальный DNS сервер (`127.0.0.1`)
2. Windows начинает использовать его
3. Все сайты:

   * разрешённые → через Google DNS (8.8.8.8)
   * остальные → блокируются
